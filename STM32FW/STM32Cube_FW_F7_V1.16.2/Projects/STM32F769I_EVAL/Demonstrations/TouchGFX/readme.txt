@par Demo Description

This is a demonstration based on TouchGFX framework for the STM32F769I Evaluation board
with 800x480 LCDs attached.

Requirements:
	Resolution: 800 * 480
	BPP: 24

The demonstration features the following modules:
    - Bird Eat Coin Game
    - 2048 - Game
    - Audio player
    - MJPEG Video player
    - Home Automation
    - Date and Clock
    - Hardware integration (Potentiometer and LCD brightness)

@par Keywords

Demonstration, TouchGFX

@par How to use it ?

Once the application built, please use ST-Link Utility or equivalent to flash the corresponding HEX file into the STM32 target.

The different assets used by the demo (fonts, images, etc.) shall be placed in an external Flash memory. For this purpose,
click on "External Loader" from the bar menu then check "MT25QL512A_STM32769I-EVAL" box before downloading the HEX file into the STM32 target.
