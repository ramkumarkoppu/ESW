@par Description

This is a demonstration for STM32F7508 MCUs with 480x272 LCDs attached.

Requirements:
	Resolution: 480 * 272
	BPP: 16

The demonstration features the following modules:
    - Custom Controls
    - Easing Equations
    - TouchGFX Presentation
    - Graphs and Pie Charts

@par Keywords

Demonstration, TouchGFX

@par How to use it ?

Please use STM32CubeProgrammer utility to flash the application's HEX file to the STM32 target.
