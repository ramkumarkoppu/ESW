@par Demo Description

@par Description

This is a demonstration based on TouchGFX framework for the STM32F746G Discovery board
with 480x272 LCDs attached.

Requirements:
	Resolution: 480 * 272
	BPP: 16

The demonstration features the following modules:
    - Custom Controls
    - Easing Equations
    - TouchGFX Presentation
    - Graphs and Pie Charts

@par Keywords

Demonstration, TouchGFX

@par How to use it ?

Once the application built, please use ST-Link Utility to flash the corresponding HEX file to the STM32 target.
