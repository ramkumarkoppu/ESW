@par Demo Description

This is a demonstration based on TouchGFX framework for the STM32756G Evaluation board
with 640x480 LCDs attached.

Requirements:
	Resolution: 640 * 480
	BPP: 16

The demonstration features the following modules:
    - 2048 - Game
    - Graph Demo
    - Bird Eat Coin Game
    - TouchGFX Presentation Demo
    - Custom Controls Demo

@par Keywords

Demonstration, TouchGFX

@par How to use it ?

Once the application built, please use ST-Link Utility to flash the corresponding HEX file to the STM32 target.
